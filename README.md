1. What did you learn about how WordPress stores its files and data?
- WordPress keeps files in folders like wp-content in XAMPP htdocs. While  pages and posts goes into a MySQL database with tables like wp_posts, seen in phpMyAdmin.
2. Which part was more complex: handling files or managing the database?
- Managing the database is harder because it needs exports, special fixes when moving from local to server.

3. Why is version control useful for a WordPress project in an office setting?
- Version control helps office teams see who changed something. Can also undo mistakes fast if an update breaks the site.
